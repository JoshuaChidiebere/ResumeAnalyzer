# AI-Powered Resume Analyzer

**Portfolio Project 5** — Parses PDF/DOCX resumes, extracts skills & experience, matches against job descriptions, and provides AI-generated feedback & scores.

## Features
- Supports PDF and DOCX uploads
- Text extraction with PyPDF2 + python-docx
- Keyword & rule-based parsing
- Optional OpenAI GPT for smart analysis, match scoring, and suggestions
- Professional PDF report generation
- Streamlit interface

## Technologies
Python • Streamlit • PyPDF2 • python-docx • pandas • OpenAI (optional) • fpdf2

## How to Run
1. `pip install -r requirements.txt`
2. `streamlit run app.py`
3. (Optional) Add your OpenAI API key for enhanced AI analysis
